01

Calcular a média de um aluno

funções com parâmetros e return
🟢 Fácil

O Colégio Byte avalia os alunos em três bimestres. A Profª. Ada precisa de uma função que calcule a média final de um aluno a partir das três notas. O cálculo usa pesos diferentes para cada bimestre.

Regra de cálculo
BimestrePesoObservação
1º Bimestre2Peso menor — início do ano
2º Bimestre3Peso intermediário
3º Bimestre5Peso maior — avaliação final

Fórmula: média = (n1×2 + n2×3 + n3×5) / 10

O que a função deve fazer
  • Receber três parâmetros: nota1, nota2, nota3 (todos float)
  • Retornar a média ponderada como float, arredondada para 1 casa decimal
  • Se qualquer nota for menor que 0 ou maior que 10, retornar -1.0 indicando entrada inválida
Exemplo de uso esperado
chamada da função python 
media = calcular_media(7.0, 8.0, 9.0)
print(f"Média: {media}")

media2 = calcular_media(5.0, 6.0, 4.0)
print(f"Média: {media2}")

invalida = calcular_media(5.0, 11.0, 8.0)
print(f"Média: {invalida}")
Saída esperada
saída no terminal
Média: 8.3 Média: 4.8 Média: -1.0
💡
DicaUse a função round(valor, 1) para arredondar para uma casa decimal antes de retornar o resultado.
Gabarito
▶ Ver solução
solução — exercício 01 python 
def calcular_media(nota1: float, nota2: float, nota3: float) -> float:
    """Calcula a média ponderada (pesos 2, 3, 5).
    Retorna -1.0 se alguma nota estiver fora do intervalo [0, 10].
    """
    for nota in [nota1, nota2, nota3]:
        if nota < 0 or nota > 10:
            return -1.0
    return round((nota1 * 2 + nota2 * 3 + nota3 * 5) / 10, 1)

Verificar a situação escolar do aluno

parâmetro padrão · retorno múltiplo
🟢 Fácil

Com a média calculada, o próximo passo é determinar a situação escolar do aluno e verificar se ele tem direito a algum benefício acadêmico. Alunos com média acima de 9.0 recebem menção honrosa, destacada no boletim.

Regras de aprovação
MédiaSituação
≥ 7.0Aprovado ✅
≥ 5.0 e < 7.0Recuperação ⚠️
< 5.0Reprovado ❌
O que a função deve fazer
  • Receber media e verificar_honra=True (parâmetro com valor padrão)
  • Retornar uma tupla: (situacao, mensagem_honra)
  • Se verificar_honra=True e a média for ≥ 9.0, a mensagem deve ser "Menção Honrosa 🏅"; caso contrário, "" (string vazia)
Exemplo de uso esperado
chamada da função python 
sit, honra = verificar_situacao(9.2)
print(f"{sit} {honra}")

sit2, honra2 = verificar_situacao(6.1)
print(f"{sit2} {honra2}")

sit3, honra3 = verificar_situacao(3.8, verificar_honra=False)
print(f"{sit3} {honra3}")
Saída esperada
saída no terminal
Aprovado ✅ Menção Honrosa 🏅 Recuperação ⚠️ Reprovado ❌
ℹ️
Lembre-sePara retornar múltiplos valores, basta separar por vírgula no return. Python os empacota automaticamente em uma tupla que pode ser desempacotada na chamada.
Gabarito
▶ Ver solução
solução — exercício 02 python 
def verificar_situacao(media: float, verificar_honra: bool = True) -> tuple:
    """Retorna (situacao, mensagem_honra) com base na média."""
    if media >= 7.0:
        situacao = "Aprovado ✅"
    elif media >= 5.0:
        situacao = "Recuperação ⚠️"
    else:
        situacao = "Reprovado ❌"

    honra = "Menção Honrosa 🏅" if (verificar_honra and media >= 9.0) else ""
    return situacao, honra

Emitir o boletim individual do aluno

composição de funções · formatação de saída
🟡 Intermediário

A Profª. Ada quer que o sistema emita um boletim formatado para cada aluno. Você vai criar a função emitir_boletim(), que reutiliza as funções dos exercícios 01 e 02 para compor o documento completo.

O que a função deve fazer
  • Receber: nome, turma, nota1, nota2, nota3
  • Chamar calcular_media() para obter a média ponderada
  • Chamar verificar_situacao() para obter situação e menção honrosa
  • Imprimir o boletim formatado conforme a saída esperada abaixo
  • Não precisa retornar nada — apenas imprimir

Composição de funçõesemitir_boletim() não recalcula nada — ela orquestra as funções que você já criou. Se a regra de aprovação mudar, basta alterar verificar_situacao() e todos os boletins são atualizados automaticamente.

solução — exercício 03 python 
def emitir_boletim(
    nome: str,
    turma: str,
    nota1: float,
    nota2: float,
    nota3: float
) -> None:
    """Imprime o boletim formatado de um aluno."""
    media    = calcular_media(nota1, nota2, nota3)
    sit, honra = verificar_situacao(media)

    linha = "=" * 36
    sep   = "-" * 36
    print(linha)
    print("     🏫 COLÉGIO BYTE — BOLETIM")
    print(linha)
    print(f"Aluno   : {nome}")
    print(f"Turma   : {turma}")
    print(f"1º Bim  : {nota1}   2º Bim: {nota2}   3º Bim: {nota3}")
    print(sep)
    print(f"Média   : {media}")
    print(f"Situação: {sit}  {honra}")
    print(linha)

Relatório de desempenho da turma

listas como parâmetro · escopo · cálculos agregados
🟡 Intermediário

Ao final do período, a Profª. Ada quer um panorama da turma inteira: quantos alunos foram aprovados, quantos foram para recuperação, quantos reprovaram, e qual foi a maior e a menor média da turma. Crie as funções necessárias para gerar esse relatório.

O que criar
  • Função calcular_media_turma(medias) — recebe uma lista de médias e retorna a média geral da turma; retorna 0.0 se a lista for vazia
  • Função contar_situacoes(medias) — retorna uma tupla (aprovados, recuperacao, reprovados) contando quantos alunos há em cada situação
  • Função relatorio_turma(nome_turma, medias) — chama as duas anteriores e imprime o relatório completo conforme a saída esperada

chamada da função python 
medias_3a = [9.7, 7.2, 5.7, 8.1, 4.3, 6.5, 9.1, 3.8]
relatorio_turma("3ºA", medias_3a)

print()

relatorio_turma("3ºB", [])

def calcular_media_turma(medias: list) -> float:
    """Retorna a média geral da turma. Retorna 0.0 se lista vazia."""
    if not medias:
        return 0.0
    return round(sum(medias) / len(medias), 1)

def contar_situacoes(medias: list) -> tuple:
    """Retorna (aprovados, recuperacao, reprovados)."""
    aprov = recup = reprov = 0
    for m in medias:
        sit, _ = verificar_situacao(m, False)
        if "Aprovado" in sit:
            aprov += 1
        elif "Recuperação" in sit:
            recup += 1
        else:
            reprov += 1
    return aprov, recup, reprov

def relatorio_turma(nome_turma: str, medias: list) -> None:
    """Imprime o relatório consolidado de desempenho da turma."""
    titulo = f"║   📊 RELATÓRIO DA TURMA — {nome_turma}   ║"
    linha  = "╔" + "═" * 34 + "╗"
    meio   = "╠" + "═" * 34 + "╣"
    fim    = "╚" + "═" * 34 + "╝"
    print(linha)
    print(f"║   📊 RELATÓRIO DA TURMA — {nome_turma}   ║")
    print(meio)
    if not medias:
        print("  Nenhum aluno avaliado ainda.")
    else:
        media_g        = calcular_media_turma(medias)
        aprov, rec, rep = contar_situacoes(medias)
        print(f"  Alunos avaliados : {len(medias)}")
        print(f"  Média da turma   : {media_g}")
        print(f"  Maior média      : {max(medias)}")
        print(f"  Menor média      : {min(medias)}")
        print("  ─────────────────────────────── ")
        print(f"  Aprovados        : {aprov}")
        print(f"  Recuperação      : {rec}")
        print(f"  Reprovados       : {rep}")
    print(fim)

Sistema de recuperação e boletim final

integração · boas práticas · mini-sistema
🔴 Desafiador

Alunos em recuperação fazem uma prova final. A nova média é calculada como (média_anterior + nota_prova_final) / 2. Se a nova média for ≥ 5.0, o aluno é aprovado; caso contrário, é reprovado. A Profª. Ada quer um boletim final que mostre todo esse processo.

O que criar
  • Função calcular_media_final(media_anterior, nota_final) — retorna a nova média calculada, arredondada para 1 casa
  • Função precisa_recuperacao(media) — retorna True se a média estiver entre 5.0 (inclusive) e 7.0 (exclusive)
  • Função emitir_boletim_final(...) — recebe nome, turma, nota1, nota2, nota3 e nota_final=None; imprime o boletim completo integrando todas as funções criadas anteriormente
ℹ️
Reutilize tudoemitir_boletim_final() deve chamar calcular_media(), precisa_recuperacao(), calcular_media_final() e verificar_situacao(). Não recalcule nada do zero.

    nota_final=7.0
)

def precisa_recuperacao(media: float) -> bool:
    """Retorna True se a média exige prova de recuperação."""
    return 5.0 <= media < 7.0

def calcular_media_final(media_anterior: float, nota_final: float) -> float:
    """Retorna a média após a prova de recuperação."""
    return round((media_anterior + nota_final) / 2, 1)

def emitir_boletim_final(
    nome: str,
    turma: str,
    nota1: float,
    nota2: float,
    nota3: float,
    nota_final: float = None
) -> None:
    """Emite o boletim final com recuperação quando aplicável."""
    media    = calcular_media(nota1, nota2, nota3)
    sit, _    = verificar_situacao(media, False)
    em_recup = precisa_recuperacao(media)

    linha = "=" * 36
    sep   = "-" * 36
    print(linha)
    print("   🏫 COLÉGIO BYTE — BOLETIM FINAL")
    print(linha)
    print(f"Aluno   : {nome}")
    print(f"Turma   : {turma}")
    print(f"1º Bim  : {nota1}   2º Bim: {nota2}   3º Bim: {nota3}")

    if em_recup and nota_final is not None:
        media_fin    = calcular_media_final(media, nota_final)
        sit_fin, _   = verificar_situacao(media_fin, False)
        print(f"Média   : {media}   → {sit}")
        print(sep)
        print(f"Prova Final    : {nota_final}")
        print(f"Média Final    : {media_fin}")
        print(f"Situação Final : {sit_fin}")
    else:
        sit_fin, honra = verificar_situacao(media)
        print(f"Média   : {media}")
        print(sep)
        print(f"Situação Final : {sit_fin}  {honra}")

    print(linha)
