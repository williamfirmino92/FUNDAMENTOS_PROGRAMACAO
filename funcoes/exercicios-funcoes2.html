01

Calcular o IMC do paciente

funções com parâmetros e return
🟢 Fácil

O primeiro passo da triagem é calcular o Índice de Massa Corporal (IMC) do paciente. A fórmula é simples: o peso (em kg) dividido pela altura (em metros) ao quadrado.

Fórmula

Fórmula: imc = peso_kg / (altura_m ** 2)

O que a função deve fazer
  • Receber dois parâmetros: peso_kg e altura_m (ambos float)
  • Retornar o IMC como float, arredondado para 1 casa decimal
  • Se peso_kg ou altura_m forem menores ou iguais a zero, retornar -1.0 indicando entrada inválida
Exemplo de uso esperado
chamada da função python 
imc = calcular_imc(70, 1.75)
print(f"IMC: {imc}")

imc2 = calcular_imc(95, 1.70)
print(f"IMC: {imc2}")

invalido = calcular_imc(-5, 1.70)
print(f"IMC: {invalido}")
Saída esperada
saída no terminal
IMC: 22.9 IMC: 32.9 IMC: -1.0
💡
DicaUse a função round(valor, 1) para arredondar o resultado antes de retornar.
Gabarito
▶ Ver solução
solução — exercício 01 python 
def calcular_imc(peso_kg: float, altura_m: float) -> float:
    """Calcula o IMC do paciente.
    Retorna -1.0 se peso ou altura forem inválidos (<= 0).
    """
    if peso_kg <= 0 or altura_m <= 0:
        return -1.0
    return round(peso_kg / (altura_m ** 2), 1)

02

Classificar IMC e verificar risco

parâmetro padrão · retorno múltiplo
🟢 Fácil

Com o IMC calculado, a clínica precisa saber em qual faixa de classificação o paciente se encontra, seguindo a tabela usada pela equipe médica. Pacientes com obesidade devem receber um alerta de risco cardiovascular.

Tabela de classificação
IMCClassificação
< 18.5Abaixo do peso
18.5 a 24.9Peso normal ✅
25.0 a 29.9Sobrepeso ⚠️
≥ 30.0Obesidade ❌
O que a função deve fazer
  • Receber imc e verificar_risco=True (parâmetro com valor padrão)
  • Retornar uma tupla: (classificacao, alerta)
  • Se verificar_risco=True e imc >= 30.0, o alerta deve ser "Risco cardiovascular elevado 🚨"; caso contrário, "" (string vazia)
Exemplo de uso esperado
chamada da função python 
classif, alerta = classificar_imc(22.9)
print(f"{classif} {alerta}")

classif2, alerta2 = classificar_imc(32.9)
print(f"{classif2} {alerta2}")

classif3, alerta3 = classificar_imc(27.3, verificar_risco=False)
print(f"{classif3} {alerta3}")
Saída esperada
saída no terminal
Peso normal ✅ Obesidade ❌ Risco cardiovascular elevado 🚨 Sobrepeso ⚠️
ℹ️
Lembre-sePara retornar múltiplos valores, basta separar por vírgula no return. Python os empacota automaticamente em uma tupla que pode ser desempacotada na chamada.
Gabarito
▶ Ver solução
solução — exercício 02 python 
def classificar_imc(imc: float, verificar_risco: bool = True) -> tuple:
    """Retorna (classificacao, alerta) com base no IMC."""
    if imc < 18.5:
        classificacao = "Abaixo do peso"
    elif imc < 25.0:
        classificacao = "Peso normal ✅"
    elif imc < 30.0:
        classificacao = "Sobrepeso ⚠️"
    else:
        classificacao = "Obesidade ❌"

    alerta = "Risco cardiovascular elevado 🚨" if (verificar_risco and imc >= 30.0) else ""
    return classificacao, alerta

Emitir a ficha de triagem do paciente

composição de funções · formatação de saída
🟡 Intermediário

A recepção da clínica precisa imprimir uma ficha de triagem para cada paciente. Você vai criar a função emitir_ficha(), que reutiliza as funções dos exercícios 01 e 02 para compor o documento completo.

O que a função deve fazer
  • Receber: nome, idade, peso e altura
  • Chamar calcular_imc() para obter o IMC
  • Chamar classificar_imc() para obter a classificação e o alerta
  • Imprimir a ficha formatada conforme a saída esperada abaixo
  • Se houver alerta, ele deve aparecer ao lado da classificação; se não houver, a linha deve mostrar apenas a classificação
  • Não precisa retornar nada — apenas imprimir
Exemplo de uso esperado
chamada da função python

print()
emitir_ficha("João Pedro", 45, 95, 1.70)

saída no terminal

💡
Composição de funçõesemitir_ficha() não recalcula nada — ela orquestra as funções que você já criou. Se a fórmula do IMC mudar, basta alterar calcular_imc() e todas as fichas são atualizadas automaticamente.

▶ Ver solução
solução — exercício 03 python 
def emitir_ficha(
    nome: str,
    idade: int,
    peso: float,
    altura: float
) -> None:
    """Imprime a ficha de triagem formatada do paciente."""
    imc            = calcular_imc(peso, altura)
    classif, alerta = classificar_imc(imc)

    linha = "=" * 36
    sep   = "-" * 36
    print(linha)
    print("   🏥 CLÍNICA VIDA+ — TRIAGEM")
    print(linha)
    print(f"Paciente : {nome}")
    print(f"Idade    : {idade} anos")
    print(f"Peso/Alt : {peso} kg / {altura} m")
    print(sep)
    print(f"IMC      : {imc}")
    if alerta:
        print(f"Classificação: {classif}  {alerta}")
    else:
        print(f"Classificação: {classif}")
    print(linha)

Relatório diário da clínica

listas como parâmetro · escopo · cálculos agregados
🟡 Intermediário

Ao final do turno, o Dr. Marcos quer um panorama dos atendimentos: quantos pacientes foram triados, qual o IMC médio do dia, e quantos pacientes caem em cada faixa de classificação. Crie as funções necessárias para gerar esse relatório.

O que criar
  • Função calcular_imc_medio(lista_imcs) — recebe uma lista de valores de IMC e retorna a média geral, arredondada para 1 casa; retorna 0.0 se a lista for vazia
  • Função contar_classificacoes(lista_imcs) — retorna uma tupla (abaixo, normal, sobrepeso, obesidade) contando quantos pacientes há em cada faixa
  • Função relatorio_diario_clinica(nome_unidade, lista_imcs) — chama as duas anteriores e imprime o relatório completo conforme a saída esperada

chamada da função python 
imcs_manha = [22.9, 32.9, 19.5, 27.3, 24.0, 35.1, 21.0]
relatorio_diario_clinica("Manhã", imcs_manha)

print()

relatorio_diario_clinica("Tarde", [])
Saída esperada
saída no terminal
╔══════════════════════════════════╗ ║ 📊 RELATÓRIO — TURNO: MANHÃ ║ ╠══════════════════════════════════╣ Pacientes triados : 7 IMC médio : 26.1 ─────────────────────────────── Abaixo do peso : 0 Peso normal : 4 Sobrepeso : 1 Obesidade : 2 ╚══════════════════════════════════╝ ╔══════════════════════════════════╗ ║ 📊 RELATÓRIO — TURNO: TARDE ║ ╠══════════════════════════════════╣ Nenhum paciente triado ainda. ╚══════════════════════════════════╝

solução — exercício 04 python 
def calcular_imc_medio(lista_imcs: list) -> float:
    """Retorna a média de IMC. Retorna 0.0 se a lista for vazia."""
    if not lista_imcs:
        return 0.0
    return round(sum(lista_imcs) / len(lista_imcs), 1)

def contar_classificacoes(lista_imcs: list) -> tuple:
    """Retorna (abaixo, normal, sobrepeso, obesidade)."""
    abaixo = normal = sobrepeso = obesidade = 0
    for imc in lista_imcs:
        classif, _ = classificar_imc(imc, False)
        if classif == "Abaixo do peso":
            abaixo += 1
        elif "normal" in classif:
            normal += 1
        elif "Sobrepeso" in classif:
            sobrepeso += 1
        else:
            obesidade += 1
    return abaixo, normal, sobrepeso, obesidade

def relatorio_diario_clinica(nome_unidade: str, lista_imcs: list) -> None:
    """Imprime o relatório consolidado de triagens do turno."""
    linha = "╔" + "═" * 34 + "╗"
    meio  = "╠" + "═" * 34 + "╣"
    fim   = "╚" + "═" * 34 + "╝"
    print(linha)
    print(f"║  📊 RELATÓRIO — TURNO: {nome_unidade.upper()}     ║")
    print(meio)
    if not lista_imcs:
        print("  Nenhum paciente triado ainda.")
    else:
        media = calcular_imc_medio(lista_imcs)
        abaixo, normal, sobrepeso, obesidade = contar_classificacoes(lista_imcs)
        print(f"  Pacientes triados : {len(lista_imcs)}")
        print(f"  IMC médio         : {media}")
        print("  ─────────────────────────────── ")
        print(f"  Abaixo do peso    : {abaixo}")
        print(f"  Peso normal       : {normal}")
        print(f"  Sobrepeso         : {sobrepeso}")
        print(f"  Obesidade         : {obesidade}")
    print(fim)

Sistema de priorização de atendimento

integração · boas práticas · mini-sistema
🔴 Desafiador

A triagem da Clínica Vida+ também mede a pressão arterial do paciente. Com base no IMC e na pressão, o sistema deve definir automaticamente a prioridade de atendimento. Neste exercício você vai integrar tudo o que foi desenvolvido nos exercícios anteriores.

Tabela de classificação de pressão
Sistólica / DiastólicaClassificação
≥ 180 ou ≥ 120Crise Hipertensiva 🚨
≥ 140 ou ≥ 90Hipertensão ⚠️
≥ 130 ou ≥ 80Pressão Elevada
Abaixo desses valoresNormal ✅
O que criar
  • Função verificar_pressao(sistolica, diastolica) — retorna a classificação da pressão conforme a tabela (verifique as condições na ordem, da mais grave para a mais leve)
  • Função definir_prioridade(classif_imc, classif_pressao) — retorna "URGENTE 🚨" se houver crise hipertensiva; "Prioritário ⚠️" se houver obesidade ou hipertensão; caso contrário, "Normal"
  • Função emitir_ficha_completa(...) — recebe nome, idade, peso, altura, sistolica e diastolica, e imprime a ficha completa com IMC, pressão e prioridade
ℹ️
Reutilize tudoemitir_ficha_completa() deve chamar calcular_imc(), classificar_imc(), verificar_pressao() e definir_prioridade(). Não recalcule nada do zero.

def verificar_pressao(sistolica: float, diastolica: float) -> str:
    """Classifica a pressão arterial do paciente."""
    if sistolica >= 180 or diastolica >= 120:
        return "Crise Hipertensiva 🚨"
    elif sistolica >= 140 or diastolica >= 90:
        return "Hipertensão ⚠️"
    elif sistolica >= 130 or diastolica >= 80:
        return "Pressão Elevada"
    else:
        return "Normal ✅"

def definir_prioridade(classif_imc: str, classif_pressao: str) -> str:
    """Define a prioridade de atendimento do paciente."""
    if "Crise" in classif_pressao:
        return "URGENTE 🚨"
    elif "Obesidade" in classif_imc or "Hipertensão" in classif_pressao:
        return "Prioritário ⚠️"
    else:
        return "Normal"

def emitir_ficha_completa(
    nome: str,
    idade: int,
    peso: float,
    altura: float,
    sistolica: float,
    diastolica: float
) -> None:
    """Emite a ficha completa de triagem com prioridade de atendimento."""
    imc             = calcular_imc(peso, altura)
    classif_imc, alerta = classificar_imc(imc)
    classif_pressao = verificar_pressao(sistolica, diastolica)
    prioridade      = definir_prioridade(classif_imc, classif_pressao)

    linha = "=" * 36
    sep   = "-" * 36
    print(linha)
    print("  🏥 CLÍNICA VIDA+ — FICHA COMPLETA")
    print(linha)
    print(f"Paciente   : {nome}")
    print(f"Idade      : {idade} anos")
    print(f"IMC        : {imc} — {classif_imc}")
    if alerta:
        print(f"Alerta IMC : {alerta}")
    print(f"Pressão    : {sistolica:.0f}/{diastolica:.0f} mmHg — {classif_pressao}")
    print(sep)
    print(f"PRIORIDADE : {prioridade}")
    print(linha)
